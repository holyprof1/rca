Privacy for the static site build.

This GitHub-hosted version is a lightweight static presentation. It does not run a live account system or a custom tracking stack inside this repo.

What we collect

Basic browser requests only

When this site is hosted on GitHub Pages, standard hosting logs may include technical request data such as IP address, browser type and time of access. This static build itself does not store form submissions in the repository.

Embedded services

Third-party resources

Some pages load external fonts, images or translation resources. Those services may process request information according to their own policies.

Your choices

How to minimize data sharing

You can block third-party requests in your browser, disable JavaScript, or avoid submitting information through external tools linked from the site.
